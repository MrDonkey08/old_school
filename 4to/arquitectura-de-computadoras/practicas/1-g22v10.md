### Simplificación con Álgebra de Boole

$$\begin{array}{l}
A = EA_0 \overline{A_1} \\
B = E(A_0 \overline{A_1} + A_1 A_0) = EA_0 \\
C = B + C = E(A_0 + \overline{A_0} A_1) = EA_0 + EA_1\\
D = E(\overline{A_0} A_1 + A_0 \overline{A_1}) = E(A_0 \oplus A_1) \\
\text{E} = C = EA_0 + EA_1 \\
F = EA_0 \\
G = E A_1
\end{array}$$
